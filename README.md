# Mongoxy

A Rust-based MongoDB proxy server for pooling and routing database connections between multiple clients and a MongoDB cluster.

## Features

- [x] Connection pooling
- [ ] Automatic connection pool resizing
- [ ] Authentication
- [ ] TLS
- [ ] Monitoring
- [ ] Logging
- [ ] Connection routing

## Usage

```bash
$ cargo run
```
